1. Данное приложение разработано на платформе Node.js, с применением технологии ajax и фреймворка bootstrap 4.

2. Данные хранятся в формате json в файле data.json. При клике на кнопку "получить данные" отправляется ajax запрос по маршруту "/get". Node.js в ответ на запрос прочитывает файл с данными, и отправляет на клиент массив объектов. На клиенте в свою очередь происходит генерация html-разметки.

3. При клике на кнопке "удалить" выполняется удаление из массива объекта с порядковым числом равным id кнопки. Так же выполняется отправка нового массива на сервер. Где тот переписывает файл data.json, и возвращает его на клиент.

4. По клику на кнопку "отправить" происходить запись в объект всех значений из инпутов. Выполняется запрос по маршуту "/send", и на сервер отправляется сформированный объект. Node.js добавляет данный объект в массив данных, записывает его в файл data.json, и отправляет обратно на клиент.

5. Так же все запросы по "/send", вместе с содержимым инпутов записываются в log файл.

6. Исходный код можно посмотреть по ссылке:https://github.com/footzi/address-book.
